# 数字花园博客 — 使用教程

## 访问方式

任意设备浏览器打开 `https://blog-7mr.pages.dev/` 即可访问。

---

## 五大模块

### 博客

- **新建文章**: 博客列表 → 点"新建" → 输入密码 → 写 Markdown → Ctrl+S 保存
- **编辑**: 打开文章 → 点"编辑" → 修改 → 保存
- **删除**: 打开文章 → 点"删除" → 确认
- **搜索**: 博客列表顶部的搜索框输入关键词
- **标签筛选**: 点击标签药片过滤，点击"全部"清除

### 日记

- **写日记**: 日记页 → 点"新建" → 输入密码 → 写内容 → 保存
- **查看**: 点击日期进入日记详情
- **编辑**: 日记详情 → 点"编辑"

### 待办

- **添加**: 输入文字 → 选优先级 → 可选截止日期 → 点"添加"
- **完成**: 点击圆形复选框勾选
- **删除**: 点右侧垃圾桶图标，支持撤销
- **快捷键**: 按 `N` 聚焦输入框

### 日程

- **添加**: 输入标题 → 选日期/时间/时长 → 点"添加"
- **查看**: 按日期分组，今天标记为绿色
- **删除**: 点右侧垃圾桶图标

### 体重

- **添加**: 输入体重 + 日期 + 备注 → 点"添加"
- **图表**: 7天/30天/90天/全部 切换，鼠标悬停查看详情
- **设置**: 点"设置"按钮，输入身高 → BMI 自动计算，输入目标体重 → 目标线显示

## 跨设备同步

所有数据存储在云端 D1 数据库，任何设备打开同一网址即可看到相同内容。无需 git push，无需导出导入。

## 日常场景

| 场景 | 操作 |
|---|---|
| **通勤记待办** | 手机打开 → 待办 → 添加任务 |
| **睡前写日记** | 手机打开 → 日记 → 点"新建" |
| **桌面写博客** | 电脑打开 → 博客 → 点"新建" → Markdown 编辑器 |
| **称完记录体重** | 手机打开 → 体重 → 输入数字 |

## 密码管理

- 默认密码: `blog2026`（尽快修改）
- 修改密码: 点击 Header 齿轮图标 → `/setup` 页面
- 写操作（新建/编辑/删除）需要密码，读操作不需要
- 密码存在浏览器 sessionStorage，关浏览器即忘

---

## 暗色/亮色切换

点击 Header 右侧的月亮/太阳图标切换主题。偏好保存在浏览器本地。

## PWA 安装

### iPhone (Safari)
1. 打开博客链接 → 点底部"分享"按钮
2. 往下滑 → 点"添加到主屏幕" → 确认

### Android (Chrome)
1. 打开博客链接 → 点右上角三个点
2. 点"添加到主屏幕" → 确认

安装后像 App 一样打开，支持离线访问已看过的页面。
